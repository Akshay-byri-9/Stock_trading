# Stock Trading Platform

A Full-stack stock trading platofrm inspired by Zerodha.

# Tech stack

-React
-Node.js
-Express.js
-Mongodb

# Features 
-User Authentication
-Dashboard UI
-Holdings and Positions
-Buy/Sell interfaces
-Responsive Design


# Installation
```bash
npm install
```

## Run Frontend

```bash
npm start
```

## Run Backend

```bash
node index.js
```
